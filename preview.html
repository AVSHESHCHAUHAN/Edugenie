Can you explain how a binary search algorithm works?

EduGenie

Binary search finds a value in a sorted list by repeatedly halving the search range:

  • Look at the middle element
  • If it matches, you're done
  • If the target is smaller, search the left half
  • If it's larger, search the right half

It runs in O(log n) time, so doubling the list size only adds one extra step.

def binary_search(arr, target):
    lo, hi = 0, len(arr) - 1
    while lo <= hi:
        mid = (lo + hi) // 2
        if arr[mid] == target:
            return mid
        elif arr[mid] < target:
            lo = mid + 1
        else:
            hi = mid - 1
    return -1
